# Portfolio

Min personliga portfolio — byggd med ren HTML, CSS och JavaScript, utan ramverk eller build-steg.

**Live:** [jonathanbackstrom.github.io/Portfolio](https://jonathanbackstrom.github.io/Portfolio/)

## Funktioner

- Mörkt tema med responsiv design (mobil, surfplatta, desktop)
- Scroll-animationer via `IntersectionObserver`
- Projektkort med tech-taggar och länkar till GitHub-repon
- Hostas via GitHub Pages

## Teknik

HTML · CSS (custom properties, grid, flexbox) · Vanilla JavaScript
